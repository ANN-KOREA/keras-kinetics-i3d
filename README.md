# keras-kinetics-i3d
Keras implementation of inflated 3d from Quo Vardis paper + weights

# Usage
```
python main.py

or

[For help]
python main.py -h
```

# Requirements
- Keras
- Tensorflow (tested) or Theano (note tested) or CNTK (not tested)
- h5py
